# TODO APP

## Getting Started
### 1. Clone the Repository
```bash
git clone https://github.com/jvconcepcion/todo-app.git
cd todo-app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Then this will be available at [http://localhost:5173](http://localhost:5173).
